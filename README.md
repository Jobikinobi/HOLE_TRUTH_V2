# The Hole Truth V2

Version two of theholetruth.org - A modern investigative journalism platform built with React and TailwindCSS.

![Website Preview](https://github.com/user-attachments/assets/387b4f05-5e45-4510-a353-716f24ea33f8)

## Features

- 🚀 **Modern Tech Stack**: Built with React 19, Vite, and TailwindCSS
- 📱 **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- 🎨 **Beautiful UI**: Dark theme with gradient backgrounds and smooth transitions
- ⚡ **Fast Performance**: Powered by Vite for lightning-fast development and builds
- 📰 **Content Sections**: Hero, About, Featured Articles, Contact, and Footer
- 🔧 **Developer Friendly**: ESLint configured, hot module replacement enabled

## Tech Stack

- **Frontend**: React 19 with functional components and hooks
- **Styling**: TailwindCSS v4 with PostCSS
- **Build Tool**: Vite 7
- **Linting**: ESLint with React plugins
- **Package Manager**: npm

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Jobikinobi/HOLE_TRUTH_V2.git
cd HOLE_TRUTH_V2
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint to check code quality
